# Scripts

This folder contains utility scripts for testing and development.

## What's In Here

- **`test-api-endpoint.js`** - Node.js script to test API endpoints
- **`test-api-endpoint.sh`** - Shell script to test API endpoints

## Usage

### Test API Endpoints
```bash
# Make sure backend server is running
cd scripts
node test-api-endpoint.js
# or
./test-api-endpoint.sh
```

## Purpose

These scripts help developers:
- Test API endpoints quickly
- Verify functionality after changes
- Debug issues with the API
